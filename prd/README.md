# Linky - 智能书签管理工具

## 项目概述

Linky 是一个遵循 Apple 设计哲学的极简书签管理工具。采用单色美学设计，专注于功能本身，让用户能够高效管理数字资源而不被分散注意力。

## 核心特性

- 🎨 **Apple 风格设计** - 纯黑背景配合系统蓝点缀，极致简约
- ⚡ **全局命令面板** - ⌘K 快速搜索所有内容
- 📱 **响应式布局** - 完美适配桌面和移动设备
- 🏷️ **统一内容管理** - 书签、文件、笔记一体化管理
- 🌙 **深色主题** - 默认深色模式，保护眼睛
- ⌨️ **键盘友好** - 完整的键盘导航支持

## 设计理念

### 单色美学

- 90% 黑白灰色调，10% 蓝色点缀
- 通过明度区分视觉层次，而非颜色
- 克制的设计，让内容成为主角

### 功能优先

- 每个元素都有其存在的必要
- 没有多余的装饰和动画
- 专注于提升工作效率

## 界面说明

### 入口页 (index.html)

- 极简设计，仅展示产品名称和描述
- 一键进入主应用
- 支持明暗主题切换

### 主应用 (bookmarks.html)

- **侧边栏导航** - 快速切换内容类型
  - All (全部)
  - Links (书签)
  - Files (文件)
  - Notes (笔记)
  - Tags (标签)
- **全局搜索** - ⌘K 唤出命令面板
- **统一卡片** - 所有内容使用一致的卡片展示
- **右键菜单** - 快速编辑、复制、删除

## 技术实现

### 设计系统

```css
/* 深色主题（默认） */
--background: #000000; /* 纯黑背景 */
--surface: #1c1c1e; /* 苹果深灰 */
--primary: #007aff; /* 系统蓝 */

/* 文本层级 */
--foreground: #ffffff; /* 主文本 */
--foreground-secondary: #ebebf5; /* 次级文本 */
--foreground-muted: #8e8e93; /* 辅助文本 */
```

### 核心功能

- **数据存储** - 使用 LocalStorage 持久化
- **响应式设计** - CSS Grid + Flexbox
- **键盘导航** - 完整的快捷键支持
- **平滑动画** - 150ms 快速过渡

## 开发笔记

### 设计决策

1. **为什么选择单色设计？**
   - 减少视觉干扰，让用户专注于内容
   - 营造专业、高级的视觉感受
   - 符合 Apple 的设计哲学

2. **为什么简化为双页面？**
   - 减少页面跳转，提升效率
   - 单页面应用更现代
   - 降低维护复杂度

3. **为什么统一管理所有内容？**
   - 书签、文件、笔记本质都是信息
   - 统一的交互降低学习成本
   - 标签系统比分类更灵活

### 性能优化

- 单文件结构，减少 HTTP 请求
- 使用 CDN 加速资源加载
- CSS 动画使用 transform 和 opacity
- 懒加载图片资源
